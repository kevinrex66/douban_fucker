# 豆瓣专辑上传工具 - Web 可视化界面

一个简洁美观的网页界面，让你无需任何命令行操作，只需输入专辑名称即可自动搜索、选择并上传到豆瓣。

## 功能特点

- 🎵 **智能搜索** - 自动从 MusicBrainz 和 Apple Music 搜索专辑
- 🖼️ **封面预览** - 搜索结果显示专辑封面
- ⚡ **一键上传** - 选择专辑后自动保存并上传豆瓣
- 🔒 **安全登录** - 首次使用引导登录豆瓣

## 快速开始

### 方式一：双击运行（推荐）

```bash
cd douban_fucker_gui
./start.sh
```

### 方式二：Python 直接运行

```bash
cd douban_fucker_gui
pip install -r requirements.txt
python main.py
```

启动后浏览器会自动打开 http://127.0.0.1:18901

## 使用流程

1. **首次使用**：点击"登录豆瓣"按钮，在浏览器中完成登录
2. **搜索专辑**：在搜索框输入专辑名或艺术家名
3. **选择专辑**：点击搜索结果中的专辑卡片
4. **自动上传**：确认后自动保存到本地并上传豆瓣

## 系统要求

- Python 3.9+
- 已安装 [douban_fucker](../douban_fucker/) 项目的依赖
- Chrome/Chromium 浏览器（用于豆瓣登录和上传）
